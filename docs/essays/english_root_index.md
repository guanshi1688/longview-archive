# Essays

This section contains the public essay archive of Longview Archive.

Longview Archive begins from a simple principle:

> **Everything begins with productive forces.**
>
> Culture, institutions, wars, and ideas are the historical logs of how societies organize production, survival, and reproduction.

## Main Sections

- [Framework](english/)
- [Series](notes/english/)
- [中文入口](chinese/)

The English framework provides the public conceptual entrance.

The series essays apply the framework to civilizational frontiers, production systems, development, value capture, China, and technology.

The Chinese materials preserve the mother-language layer of the archive.
