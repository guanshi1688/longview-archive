# 生产的边界

“生产的边界”系列，讨论生产能力为什么不能靠外部输入自动生成。

如果说“扩张的边界”回答的是影响为什么不等于复制，那么“生产的边界”回答的是输入为什么不等于生成。

修路、建厂、通电、人口、资源、工业园、廉价劳动力和产业转移，都只是条件或接口；真正决定一个社会能不能工业化的，是它能否把这些输入转化为内部持续运转的生产闭环。
