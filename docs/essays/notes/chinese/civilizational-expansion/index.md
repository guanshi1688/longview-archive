# 文明的扩张

本目录保存中文线中关于文明扩张、制度复制、边疆承接能力、外部循环、全球南方与做工文明的札记文章。

这些文章不是 Longview Archive｜观势档案 的正本正文，而是正本形成之前的材料层、案例层与概念沉积层。

它们最初服务于中文平台发布、概念预埋与问题测试，后续可根据需要被压缩、重写或转化为正本文章、英文短文与专题合订本。

## 当前定位

- 类型：中文札记 / 平台文章 / 理论材料层
- 位置：Longview Archive｜观势档案 的札记层
- 状态：内部归档为主
- 网站导航：暂不主动展示全部文章
- 用途：为文明扩张、制度复制、外部承接、全球南方与做工文明等问题打地基

## 核心问题

一个文明的扩张边界，不是它的军队、商船、商品、宗教、资本或技术能够抵达哪里。

更深处的问题是：

一种文明的生存方式，能否被外部空间承接，并转化为当地稳定的生产方式、组织方式、收益结构和秩序系统。
