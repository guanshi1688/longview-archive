# Longview Archive｜观势档案

Longview Archive｜观势档案，是星衡｜Aster Vale 的公开文本档案。

本档案用于保存长期观察文章、理论核心术语与星衡札记。

本站为公开标准正本。其他平台如有节选、改写或传播版本，若内容存在差异，以本站版本为准。

## 当前结构

- Core Terms｜理论核心术语
- Longview Essays｜长线观势文章
- Aster Vale Notes｜星衡札记
- Version Records｜版本记录

## 公开边界

本站只发布可公开传播的阳面文本。

草稿、私密札记、未完成推演与非公开材料，不在本站保存。
