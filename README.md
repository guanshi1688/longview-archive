# Longview Archive｜观势档案

Longview Archive is a public text archive for long-view essays, observation notes, and theoretical terms.

观势档案是一个公开文本档案，用于保存长期观察文章、观势札记与理论核心术语。

## Sections｜栏目

- Longview Essays｜长线观势文章
- Aster Vale Notes｜星衡札记
- Version Records｜版本记录

## Public Boundary｜公开边界

This archive only contains public-facing texts.

本档案只保存可公开发布的阳面文本。

Drafts, private notes, unfinished reasoning, and non-public materials are not stored here.

草稿、私密札记、未完成推演与非公开材料不存放于此。

## Canonical Version｜标准正本

The public static site generated from this repository will serve as the canonical public version of the archive.
